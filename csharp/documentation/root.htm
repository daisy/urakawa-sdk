Urakawa Toolkit

Introduction

The Urakawa Toolkit is an SDK (Software Development Kit) that is intended to facilitate easy creation of multimedia authoring applications

Integration Requirements

The Urakawa Toolkit is implemented in C# v2.0 and makes use of the .Net Framework v2.0.

In order to use the toolkit a C# compiler supporting the .Net Framework v2.0 is needed. The toolkit has been developed and tested with the Microsoft Visual Studio 2005 Development Environment.

The toolkit can be compiled using the gmcs compiler of Mono version 1.1.13.8. This in principle means that the toolkit can be used on non-Microsoft Windows platforms like Linux and Mac OS X.

Documentation

Documentation was generated using a modified version Sandcastle

Unit testing is done using NUnit version 2.2.8

Example

The following example will load create an urakawa.core.project.Project, load a XUK file and printout the number of urakawa.media.IMedia object found in each urakawa.core.core.IChannel

C#

using urakawa;
using urakawa.core;
using urakawa.properties.channel;
using urakawa.examples;

public class Program
{
    [STAThread]
    static void Main(string[] args)
    {
        Project proj = new Project();
        proj.openXUK(new Uri(System.IO.Path.Combine(
                System.IO.Directory.GetCurrentDirectory(),
                args[0])));
        foreach (IChannel ch in proj.getPresentation().getChannelsManager().getListOfChannels())
        {
            CollectMediaFromChannelCoreNodeVisitor visitor = new CollectMediaFromChannelCoreNodeVisitor(ch);
            proj.getPresentation().getRootNode().acceptDepthFirst(visitor);
            Console.WriteLine(
                    "Channel {0} contains {1:0} media objects",
                    ch.getName(), visitor.CollectedMedia.Length);
        }
    }
}
