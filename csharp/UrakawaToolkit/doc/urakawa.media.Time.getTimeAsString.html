Time.getTimeAsString Method 

Gets a string representation of the instance

public string getTimeAsString();

Return Value

The string representation

Remarks

The format of the string representation [-][d.]hh:mm:ss[.f], where d is a number of days, hh is two-digit hours between 00 and 23, mm is two-digit minutes between 00 and 59, ss is two-digit seconds between 00 and 59 and where f is the second fraction with between 1 and 7 digits
