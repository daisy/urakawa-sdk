OperationNotValidException Class

(from the design docs) This exception should be thrown/raised when trying to call an operation (aka class method) on an object that does not allow a specific modification of the state in the current context. ... Wherever a "canDoXXX()" method can be found, the corresponding operation "doXXX()" should use this exception/error to let the user-agent of the API/Toolkit know about the non-permitted operation for which there was an attempt to execute.
