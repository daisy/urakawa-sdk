IXUKAble Methods

The methods of the IXUKAble interface are listed below. For a complete list of IXUKAble interface members, see the IXUKAble Members topic.

Public Instance Methods

XUKIn The implementation of XUKIn is expected to read and remove all tags up to and including the closing tag matching the element the reader was at when passed to it. The call is expected to be forwarded to any owned element, in effect making it a recursive read of the XUK file
XUKOut The implementation of XUKOut is expected to write a tag for the object it is called on. The call should be forwarded to any owned object, making it in effect be a recursive write of the CoreTree
