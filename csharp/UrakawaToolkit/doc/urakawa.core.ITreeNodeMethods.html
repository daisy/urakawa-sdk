ITreeNode Methods

The methods of the ITreeNode interface are listed below. For a complete list of ITreeNode interface members, see the ITreeNode Members topic.

Public Instance Methods

appendChild Appends a child ITreeNode to the end of the list of children
indexOf Gets the index of a given child ITreeNode
insertAfter Inserts a new ITreeNode child after the given child.
insertBefore Inserts a new ITreeNode child before the given child.
removeChildOverloaded. Removes the child at a given index.
replaceChildOverloaded. Replaces the child ITreeNode at a given index with a new ITreeNode
