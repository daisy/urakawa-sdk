copy The actual Property object implementations must define the semantics of such copy, as it has critical implications in terms of memory management, shared object pools, etc. e.g.: a ChannelsProperty that has Media objects pointing to actual files (like MP3 audio files) => Media should be sufficiently abstract and well-managed via the some sort of MediaAssetManager to guarantee that sharing conflicts are resolved transparently.
