Urakawa Toolkit

urakawa.media Namespace

This namespace contains the media related classes of the toolkit

Namespace hierarchy

Classes

Class Description
AudioMedia AudioMedia is the audio object. It is time-based and comes from an external source.
ClippedMedia Used internally to simplify implementation of media objects.
ExternalMedia Used internally to simplify implementation of media objects.
ImageMedia ImageMedia is the image object. It has width, height, and an external source.
MediaFactory The media factory will create any media object of MediaType.xxx
MediaLocation MediaLocation is just a string which represents a file's path This simple idea could be extended in the (near)future.
SequenceMedia SequenceMedia is a collection of same-type media objects The first object in the collection determines the collection's type.
TextMedia TextMedia represents a text string
Time The Time object represents a timestamp.
TimeDelta TimeDelta is the difference between two timestamps (Times)
VideoMedia VideoMedia is the video object. It is time-based, comes from an external source, and has a visual presence.

Interfaces

Interface Description
IAudioMedia Summary description for IAudioMedia.
IClippedMedia This interface is for referring to time-based segments of external media
IExternalMedia This interface associates a media object with its source location
IImageMedia Represents images which are external media and have a height and width
IImageSize Summary description for IImageSize.
IMedia This is the base interface for all media-related classes and interfaces. Media is continuous (time-based) or discrete (static), and is of a specific type.
IMediaFactory This is the interface to a factory which creates media objects.
IMediaLocation Simple interface used to correlate a media object to its actual file
ISequenceMedia A sequence is a collection of any media object. All objects in the collection must be of the same type.
ITextMedia Text media conforms to this interface.
ITime A simple interface for a measure of time.
ITimeDelta the difference between two ITime objects is an ITimeDelta
IVideoMedia Video media is both time-based and has a visual presence

Enumerations

Enumeration Description
MediaType Enumeration of basic media types.
